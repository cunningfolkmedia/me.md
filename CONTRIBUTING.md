# Contributing

Contributions are welcome.

Good contributions include:

- clearer spec language
- better examples
- privacy and redaction improvements
- CLI lint rules
- parser bug fixes
- additional templates
- interoperability notes for agent tools

## Principles

Keep the format:

- plain-text first
- human-readable
- machine-parseable
- privacy-aware
- useful to actual working agents
- simple enough to maintain by hand

## Development

```bash
npm install
npm test
npm run lint -- examples/cunningfolk.me.md
```

## Pull requests

Please include:

- what changed
- why it matters
- examples before/after when possible
- any schema changes
