# Agent Instructions

When working in this repository:

1. Read `README.md`, `SPEC.md`, and `me.md` before editing.
2. Keep the format plain Markdown plus YAML front matter.
3. Preserve the required `##` section order unless updating the spec and validator together.
4. Run `node bin/me-md.js lint me.md` after changes.
5. Treat `private.me.md` and `*.local.me.md` as intentionally ignored private files.
6. Do not add private contact details, API keys, or sensitive personal data to examples.
